# TC-10.6: Sign-Out Does Not Clear Persisted State

**System Requirement**: [SR-10.6](../../system-requirements/authentication.md)

## TC-10.6.1: Active filters and selected model remain unchanged after signing out

**Functional Requirements**: [FR-10.3](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- The user **John Doe** is authenticated; **Test Architecture** is loaded; the
  **ApplicationService** entity type filter is unchecked (Service X is hidden); the graph view is
  active

### Test Steps

1. Click the **Sign out** button
   - **Expected**: The user is signed out (header shows "Sign in"); the **Test Architecture** model
     remains loaded and the graph is still visible
2. Observe the Entities filter panel in the sidebar
   - **Expected**: The **ApplicationService** checkbox is still unchecked; **Service X** is still
     hidden from the graph; the filter state is unchanged from before sign-out

### Post-conditions

- Active filters and loaded model are preserved after sign-out

### Test Data

| Field            | Value                          |
| ---------------- | ------------------------------ |
| User             | John Doe                       |
| Model            | Test Architecture              |
| Filter unchecked | ApplicationService             |
| After sign-out   | Filter unchanged, model loaded |

## TC-10.6.2: Theme preference is preserved after signing out

**Functional Requirements**: [FR-10.3](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- The user **John Doe** is authenticated; the current theme is **Dark**; `localStorage` key
  `architeezy.theme` is `"dark"`

### Test Steps

1. Click the **Sign out** button
   - **Expected**: The user is signed out; the dark theme remains applied; no theme flash or reset
     to light/system occurs
2. Inspect `localStorage` for key `architeezy.theme`
   - **Expected**: The value is still `"dark"`

### Post-conditions

- Theme preference in `localStorage` is untouched by sign-out

### Test Data

| Field            | Value                 |
| ---------------- | --------------------- |
| User             | John Doe              |
| Theme            | Dark                  |
| localStorage key | architeezy.theme=dark |
| After sign-out   | Dark (unchanged)      |
