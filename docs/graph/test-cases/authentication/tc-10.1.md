# TC-10.1: Anonymous Mode

**System Requirement**: [SR-10.1](../../system-requirements/authentication.md)

## TC-10.1.1: Application loads without any token and all public models are accessible

**Functional Requirements**: [FR-10.1](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- No authentication token is present in `localStorage` or session storage
- The API exposes at least one public model: **Test Architecture**

### Test Steps

1. Open the application at `/graph/` with no stored token
   - **Expected**: The application loads normally; the "Select Model" modal appears; the header
     shows a **Sign in** button and no user name
2. Select the **Test Architecture** model
   - **Expected**: The model loads successfully; the graph renders with all nodes and edges; no
     authentication error is shown

### Post-conditions

- Public model is accessible in anonymous mode; "Sign in" button is visible in the header

### Test Data

| Field  | Value             |
| ------ | ----------------- |
| Token  | None              |
| Model  | Test Architecture |
| Header | Sign in button    |

## TC-10.1.2: Attempting to load a private model in anonymous mode shows a 401 error notification

**Functional Requirements**: [FR-10.1](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- No authentication token is present; the API has a private model **Private Architecture** that
  returns HTTP 401 for unauthenticated requests

### Test Steps

1. Open the application at `/graph/` and open the model selector
2. Select **Private Architecture** from the model list
   - **Expected**: The model selector closes; a 401 error notification appears (e.g. "Access denied.
     Please sign in to view this model."); the graph canvas remains empty; the header still shows
     **Sign in**

### Post-conditions

- Application shows an error notification; no private model data is rendered

### Test Data

| Field          | Value                |
| -------------- | -------------------- |
| Token          | None                 |
| Model          | Private Architecture |
| API response   | 401 Unauthorized     |
| Expected error | 401 notification     |
