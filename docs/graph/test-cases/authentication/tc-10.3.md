# TC-10.3: Authentication Initiation

**System Requirement**: [SR-10.3](../../system-requirements/authentication.md)

## TC-10.3.1: Clicking "Sign in" opens an authentication popup window

**Functional Requirements**: [FR-10.2](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- No authentication token is present; the application is open at `/graph/`; the browser allows popup
  windows from the application origin

### Test Steps

1. Click the **Sign in** button in the header
   - **Expected**: A new popup window opens pointing to the authentication provider; the popup
     window is distinct from the main application window (not a modal overlay); the main application
     window remains open and interactive in the background

### Post-conditions

- The authentication popup window is open; the main application is still accessible

### Test Data

| Field    | Value              |
| -------- | ------------------ |
| Token    | None               |
| Action   | Click "Sign in"    |
| Expected | Popup window opens |

## TC-10.3.2: Blocked popup shows an inline error with a fallback link

**Functional Requirements**: [FR-10.2](../../functional-requirements.md#fr-10-authentication)

### Preconditions

- No authentication token is present; the browser is configured to **block popups** from the
  application origin

### Test Steps

1. Click the **Sign in** button in the header
   - **Expected**: The popup is blocked by the browser; an inline error or notification appears in
     the application (not a browser alert) explaining that the popup was blocked and providing a
     **fallback link** to open the authentication page manually
2. Click the fallback link
   - **Expected**: The authentication page opens in the current tab or a new tab

### Post-conditions

- User receives clear guidance when the popup is blocked; authentication can still be initiated

### Test Data

| Field         | Value               |
| ------------- | ------------------- |
| Token         | None                |
| Popup setting | Blocked             |
| Expected      | Inline error + link |
