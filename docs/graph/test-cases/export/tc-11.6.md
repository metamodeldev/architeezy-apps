# TC-11.6: Optional Title/Legend in Exported Image

**System Requirement**: [SR-11.6](../../system-requirements/export.md)

## TC-11.6.1: When title/legend option is enabled, exported image includes a legend area

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the graph view is active; the "Include title/legend" option is
  available in the export settings or Graph Settings panel

### Test Steps

1. Enable the **Include title/legend** option in Graph Settings or the export dialog
2. Click the **Export Image** button and select **Export as PNG**
   - **Expected**: A PNG file is downloaded; when opened, the image includes a clearly visible
     legend area showing the element type color indicators (e.g. **ApplicationComponent**,
     **ApplicationService**) and/or the model title **Test Architecture**

### Post-conditions

- Exported image contains a legend/title area

### Test Data

| Field           | Value                                |
| --------------- | ------------------------------------ |
| Model           | Test Architecture                    |
| Legend option   | Enabled                              |
| Expected in PNG | Legend area with element type colors |
| Format          | PNG                                  |

## TC-11.6.2: When title/legend option is disabled, exported image contains no legend

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the graph view is active; the "Include title/legend" option is
  **disabled** in the export settings or Graph Settings panel

### Test Steps

1. Confirm the **Include title/legend** option is disabled
2. Click the **Export Image** button and select **Export as PNG**
   - **Expected**: A PNG file is downloaded; when opened, the image contains no legend area and no
     model title overlay — only the graph nodes and edges are visible

### Post-conditions

- Exported image contains no legend or title when the option is disabled

### Test Data

| Field           | Value             |
| --------------- | ----------------- |
| Model           | Test Architecture |
| Legend option   | Disabled          |
| Expected in PNG | No legend area    |
| Format          | PNG               |
