# TC-11.6: Exported Image Reproduces Visible Canvas Content

**System Requirement**: [SR-11.6](../../system-requirements/export.md)

## TC-11.6.1: Exported PNG includes the legend when it is visible on the canvas

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the graph view is active
- The **Legend** setting is **enabled**; the legend shape is visible on the canvas showing
  **ApplicationComponent**, **ApplicationService**, and **AssociationRelationship** entries

### Test Steps

1. Click the **Export Image** button and select **Export as PNG**
   - **Expected**: A PNG file is downloaded; when opened, the image shows the graph nodes and edges
     and also contains the legend shape with its type entries and color circles at the position it
     occupied on the canvas

### Post-conditions

- The downloaded PNG file contains the legend shape

### Test Data

| Field           | Value                                      |
| --------------- | ------------------------------------------ |
| Model           | Test Architecture                          |
| Legend          | Enabled (visible on canvas)                |
| Expected in PNG | Legend shape with type entries and circles |
| Format          | PNG                                        |

## TC-11.6.2: Exported PNG does not include the legend when it is hidden

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the graph view is active
- The **Legend** setting is **disabled**; no legend shape is present on the canvas

### Test Steps

1. Click the **Export Image** button and select **Export as PNG**
   - **Expected**: A PNG file is downloaded; when opened, the image contains only graph nodes and
     edges — no legend shape is visible

### Post-conditions

- The downloaded PNG file contains no legend shape

### Test Data

| Field           | Value             |
| --------------- | ----------------- |
| Model           | Test Architecture |
| Legend          | Disabled (hidden) |
| Expected in PNG | No legend shape   |
| Format          | PNG               |
