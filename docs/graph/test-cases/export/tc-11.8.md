# TC-11.8: Export Respects Active Filters

**System Requirement**: [SR-11.8](../../system-requirements/export.md)

## TC-11.8.1: CSV export with ApplicationService unchecked excludes Service X from the output

**Functional Requirements**: [FR-11.1](../../functional-requirements.md#fr-11-data-export),
[FR-3.1](../../functional-requirements.md#fr-3-filtering-system)

### Preconditions

- **Test Architecture** is loaded with **Component A** (ApplicationComponent), **Component B**
  (ApplicationComponent), and **Service X** (ApplicationService); the table view is active; the
  **ApplicationService** entity type is **unchecked** in the Entities filter

### Test Steps

1. Verify that **Service X** is not visible in the table (filtered out)
2. Click the **Export CSV** button
   - **Expected**: A CSV file is downloaded
3. Open the downloaded CSV file
   - **Expected**: The file contains rows for **Component A** and **Component B**; **Service X** is
     not present anywhere in the file

### Post-conditions

- Exported CSV excludes elements filtered out by the active entity type filter

### Test Data

| Field           | Value                        |
| --------------- | ---------------------------- |
| Model           | Test Architecture            |
| Filter          | ApplicationService unchecked |
| Expected in CSV | Component A, Component B     |
| Not in CSV      | Service X                    |
| Row count       | 2                            |

## TC-11.8.2: Graph image export captures only visible nodes (those not filtered out)

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export),
[FR-3.1](../../functional-requirements.md#fr-3-filtering-system)

### Preconditions

- **Test Architecture** is loaded with **Component A**, **Component B**, and **Service X**; the
  graph view is active; the **ApplicationService** entity type is **unchecked** (so **Service X**
  node is hidden from the graph canvas)

### Test Steps

1. Verify that **Service X** node is not visible on the graph canvas
2. Click the **Export Image** button and select **Export as PNG**
   - **Expected**: A PNG file is downloaded
3. Open the downloaded PNG image
   - **Expected**: The image shows **Component A** and **Component B** nodes; **Service X** is not
     present in the image; the edges connecting only the visible nodes are shown

### Post-conditions

- Exported graph image includes only nodes that are visible (not filtered out)

### Test Data

| Field           | Value                        |
| --------------- | ---------------------------- |
| Model           | Test Architecture            |
| Filter          | ApplicationService unchecked |
| Expected in PNG | Component A, Component B     |
| Not in PNG      | Service X                    |
| Format          | PNG                          |
