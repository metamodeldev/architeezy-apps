# TC-11.2: CSV Export Includes Visible Rows

**System Requirement**: [SR-11.2](../../system-requirements/export.md)

## TC-11.2.1: CSV export with filters applied includes only the visible filtered rows

**Functional Requirements**: [FR-11.1](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded with elements **Component A** (ApplicationComponent), **Component
  B** (ApplicationComponent), and **Service X** (ApplicationService); the table view is active; the
  **ApplicationService** entity type is **unchecked** in the Entities filter (so **Service X** is
  hidden)

### Test Steps

1. Observe the table — only **Component A** and **Component B** are listed
2. Click the **Export CSV** button in the table toolbar
   - **Expected**: A CSV file is downloaded; the file contains only the rows for **Component A** and
     **Component B**; **Service X** is not present in the file

### Post-conditions

- Downloaded CSV contains only the 2 visible rows matching the active filter

### Test Data

| Field           | Value                             |
| --------------- | --------------------------------- |
| Model           | Test Architecture                 |
| Filter          | ApplicationService unchecked      |
| Visible rows    | Component A, Component B          |
| Expected in CSV | Component A, Component B (2 rows) |
| Not in CSV      | Service X                         |

## TC-11.2.2: CSV export with no filters includes all elements in the model

**Functional Requirements**: [FR-11.1](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded with **Component A**, **Component B**, and **Service X**; the
  table view is active; all entity type filters are checked (nothing is filtered out)

### Test Steps

1. Verify all three elements are visible in the table
2. Click the **Export CSV** button
   - **Expected**: A CSV file is downloaded; the file contains rows for **Component A**, **Component
     B**, and **Service X** — all three elements are present

### Post-conditions

- Downloaded CSV contains all 3 elements from the model

### Test Data

| Field           | Value                               |
| --------------- | ----------------------------------- |
| Model           | Test Architecture                   |
| Filter          | None (all types checked)            |
| Expected in CSV | Component A, Component B, Service X |
| Row count       | 3                                   |
