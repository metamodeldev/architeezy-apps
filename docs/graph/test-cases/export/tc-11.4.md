# TC-11.4: Graph Image Export Accessibility

**System Requirement**: [SR-11.4](../../system-requirements/export.md)

## TC-11.4.1: An "Export Image" button is visible in the graph toolbar when graph view is active

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the graph view is active

### Test Steps

1. Observe the graph toolbar
   - **Expected**: An **Export Image** button (or icon with tooltip "Export Image") is visible in
     the graph toolbar; the button is enabled and clickable

### Post-conditions

- "Export Image" button is present and enabled in the graph toolbar

### Test Data

| Field  | Value             |
| ------ | ----------------- |
| Model  | Test Architecture |
| View   | Graph             |
| Button | Export Image      |
| State  | Enabled           |

## TC-11.4.2: "Export Image" button is disabled when no model is loaded

**Functional Requirements**: [FR-11.2](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- The application is open at `/graph/`; no model has been loaded; the graph view is shown (empty
  canvas)

### Test Steps

1. Observe the graph toolbar
   - **Expected**: The **Export Image** button is visible but in a **disabled** state (greyed out,
     not clickable, `disabled` attribute or `aria-disabled="true"` present)
2. Attempt to click the disabled button
   - **Expected**: No action is taken; no image file download is triggered; no error notification
     appears

### Post-conditions

- "Export Image" button is disabled when no model is loaded

### Test Data

| Field  | Value        |
| ------ | ------------ |
| Model  | None         |
| View   | Graph        |
| Button | Export Image |
| State  | Disabled     |
