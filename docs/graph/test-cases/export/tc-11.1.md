# TC-11.1: CSV Export Accessibility

**System Requirement**: [SR-11.1](../../system-requirements/export.md)

## TC-11.1.1: An "Export CSV" button is visible in the table toolbar when table view is active

**Functional Requirements**: [FR-11.1](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- **Test Architecture** is loaded; the table view is active

### Test Steps

1. Observe the table toolbar
   - **Expected**: An **Export CSV** button is visible in the table toolbar; the button is enabled
     and clickable

### Post-conditions

- "Export CSV" button is present and enabled in the table toolbar

### Test Data

| Field  | Value             |
| ------ | ----------------- |
| Model  | Test Architecture |
| View   | Table             |
| Button | Export CSV        |
| State  | Enabled           |

## TC-11.1.2: "Export CSV" button is disabled when no model is loaded

**Functional Requirements**: [FR-11.1](../../functional-requirements.md#fr-11-data-export)

### Preconditions

- The application is open at `/graph/`; the "Select Model" modal has been dismissed or no model has
  been selected; the table view is active (or can be navigated to)

### Test Steps

1. Navigate to the table view without loading a model
   - **Expected**: The **Export CSV** button is visible in the table toolbar but is in a
     **disabled** state (greyed out, not clickable, and has `disabled` attribute or `aria-disabled`
     set to `true`)
2. Attempt to click the disabled button
   - **Expected**: No action is taken; no CSV file is downloaded; no error notification appears

### Post-conditions

- "Export CSV" button is disabled when no model is loaded

### Test Data

| Field  | Value      |
| ------ | ---------- |
| Model  | None       |
| View   | Table      |
| Button | Export CSV |
| State  | Disabled   |
